🚀 Pasos para subir tu web a Vercel

1

Preparar base de datos en Supabase

Ve a tu panel de Supabase, entra en SQL Editor, abre una consulta en blanco e importa el archivo:

sql/supabase_setup.sql

Esto creará la tabla de votos protegida con Row Level Security.

2

Modificar las credenciales en el archivo JS

Abre el archivo JS/supabase-client.js y reemplaza las variables placeholders por tus credenciales de Supabase. Así no dependerás del almacenamiento local.

3

Crear Repositorio en GitHub

Inicializa Git en tu carpeta de proyecto y súbelo a un repositorio público o privado en GitHub:

git init
git add .
git commit -m "Migración a Supabase"
git branch -M main
git remote add origin TU_URL_DE_GITHUB
git push -u origin main
4

Conectar GitHub con Vercel

1. Ve a vercel.com y crea una cuenta.
2. Haz clic en Add New → Project.
3. Conecta tu cuenta de GitHub e importa tu repositorio recién creado.
4. Haz clic en Deploy (No necesitas configurar variables en este caso).

5

¡Listo! Obtén tu dominio y crea tu QR

Vercel te dará un dominio gratuito con extensión .vercel.app. Cópialo, colócalo arriba en el generador de QR, personalízalo y descárgalo para pegarlo en tu afiche o presentación.

6

(Opcional) Activar Google OAuth

Si deseas activar el Login con Google, en la consola de Google Cloud, agrega el dominio de Vercel en las URIs autorizadas:

https://TU_PROYECTO.vercel.app/views/catalogo.html

Y configúralo en los ajustes de Supabase → Authentication → Providers → Google.
